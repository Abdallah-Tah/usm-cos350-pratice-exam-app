Section 1: UNIX Basics & Commands (Lecture 1)

Q1 Easy

Which command displays the manual page for the grep command?

Show Explanation

Correct: B) man grep displays the manual page.

The man command (manual) is the standard way to view documentation for UNIX commands. Manual pages are organized into sections: section 1 for user commands, section 2 for system calls, section 3 for library functions.
Q2 Easy

What does the command ls -la do?

Show Explanation

Correct: A)

-l = long format (permissions, owner, size, date). -a = all files including hidden (those starting with .). Combined as -la or -al.
Q3 Medium

Which grep pattern finds words that start with "pre"?

Show Explanation

Correct: C)

^ anchors the pattern to the start of the line. $ anchors to the end. Without ^, it matches "pre" anywhere in the line.
Q4 Medium

What does grep 'ing$' words find?

Show Explanation

Correct: B)

The $ anchor matches the end of the line, so ing$ matches lines ending with "ing".
Q5 Easy

Which command shows the current working directory?

Show Explanation

Correct: A) pwd stands for "print working directory".

Section 2: Arrays & Pointers (Lecture 3)

Q6 Easy

What is wrong with this code?

int a[10];
a[10] = 5;
Show Explanation

Correct: B)

Array a[10] has indices 0-9. Accessing a[10] is out of bounds and causes undefined behavior. C does NOT check array bounds at runtime.
Q7 Medium

How do you declare a pointer to an integer?

Show Explanation

Correct: C) int *p; declares p as a pointer to int.

* in declarations means "pointer to". & is the address-of operator used in expressions, not declarations.
Q8 Hard

What is the output?

int x = 7;
int *p = &x;
printf("%d\n", *p);
Show Explanation

Correct: D) Output is 7.

p stores the address of x. *p dereferences the pointer to get the value at that address, which is 7.
Q9 Medium

What's wrong with this code?

char *p;
*p = 'A';
Show Explanation

Correct: B)

p is uninitialized and doesn't point to valid memory. Dereferencing it causes undefined behavior. Fix: char c; char *p = &c; *p = 'A';
Q10 Easy

In C, what is the array name equivalent to?

Show Explanation

Correct: A)

In most contexts, the array name a is equivalent to &a[0], the address of the first element.

Section 3: File I/O & System Calls (Lectures 4, 6)

Q11 Medium

What's the difference between open() and fopen()?

Show Explanation

Correct: B)

open() is a system call returning int (file descriptor), used with read(), write(). fopen() is a C library function returning FILE*, used with fread(), fwrite(). fopen() provides buffering.
Q12 Easy

What does read() return at end of file?

Show Explanation

Correct: B) Returns 0 at EOF, -1 on error, positive number = bytes read.

Q13 Easy

What does lseek() do?

Show Explanation

Correct: B)

lseek(fd, offset, whence) where whence can be SEEK_SET (from start), SEEK_CUR (from current), or SEEK_END (from end).
Q14 Easy

Which system call closes a file descriptor?

Show Explanation

Correct: A) close(fd) closes file descriptor. fclose() is for FILE* pointers.

Q15 Medium

What does open() return on failure?

Show Explanation

Correct: C) open() returns -1 on failure. Check with: if ((fd = open(...)) < 0)

Section 4: Strings (Lecture 5)

Q16 Easy

What marks the end of a C string?

Show Explanation

Correct: B) C strings end with '\0'. "hello" takes 6 bytes (5 chars + null terminator).

Q17 Easy

Why use int with getchar() instead of char?

Show Explanation

Correct: B)

getchar() returns all char values (0-255) plus EOF (-1). int can distinguish EOF from valid characters; char cannot.
Q18 Easy

Which function converts string "42" to integer 42?

Show Explanation

Correct: A) int n = atoi("42"); More robust: strtol()

Q19 Medium

What does strlen("hello") return?

Show Explanation

Correct: C) Returns 5. strlen() counts chars before '\0', NOT including '\0'.

Q20 Hard

What's dangerous about scanf("%s", buf) where buf[80]?

Show Explanation

Correct: D)

scanf("%s") doesn't know buffer size. Fix: scanf("%79s", buf) limits input to 79 chars + null terminator.

Section 5: File Permissions (Lecture 11)

Q21 Medium

You want: user read/write, group read, other read. Which is correct?

Show Explanation

Correct: B) 644 = rw-r--r-- (6=rw, 4=r, 4=r). r=4, w=2, x=1.

Q22 Medium

What does chmod 755 prog mean?

Show Explanation

Correct: C) 755 = rwxr-xr-x. Common for executables.

Q23 Hard

What's the octal mask for "other read" permission bit?

Show Explanation

Correct: A)

Bottom 9 bits: rwxrwxrwx. Other read is bit 2 = 0004. Other write=0002, other execute=0001.
Q24 Medium

Which system call gets file information including size and permissions?

Show Explanation

Correct: D) Both work. stat() takes filename, fstat() takes file descriptor.

Q25 Easy

Which field of struct stat stores file size in bytes?

Show Explanation

Correct: B) info.st_size contains file size in bytes.

Section 6: Buffering (Lecture 7)

Q26 Medium

Which provides buffering?

Show Explanation

Correct: A)

fread()/fwrite() are buffered (use FILE*). read()/write() are unbuffered system calls (use fd).
Q27 Medium

Why is buffering useful?

Show Explanation

Correct: B)

Buffering collects data in memory before making system calls, reducing expensive kernel transitions.
Q28 Easy

Which function forces buffered output to be written?

Show Explanation

Correct: A) fflush(fp) flushes the buffer for FILE*. fsync() is for file descriptors.

Section 9: Directories (Lectures 10, 12)

Q36 Medium

What's the difference between hard link and symbolic link?

Show Explanation

Correct: B)

Hard link: another name for same inode/data. Symbolic link: separate file containing pathname. If original deleted, hard link still works, symlink breaks.
Q37 Easy

Which command creates a symbolic link?

Show Explanation

Correct: C) ln -s creates symbolic link. ln without -s creates hard link.

Q38 Medium

What does opendir() return?

Show Explanation

Correct: A) DIR *dp = opendir(dirname). Use with readdir(), close with closedir().

Q39 Medium

Which function reads the next directory entry?

Show Explanation

Correct: B) struct dirent *entry = readdir(dp). Returns NULL at end.

Section 11: Signals (Lecture 14)

Q44 Easy

What signal is sent by Ctrl-C?

Show Explanation

Correct: B) SIGINT (interrupt). Ctrl-Z sends SIGTSTP (stop).

Q45 Easy

What's the default signal sent by kill pid?

Show Explanation

Correct: A) SIGTERM (15). Use kill -9 pid for SIGKILL.

Q46 Medium

Which signal CANNOT be caught or ignored?

Show Explanation

Correct: C) SIGKILL (9) and SIGSTOP cannot be caught, blocked, or ignored.

Q47 Medium

What does signal(SIGINT, SIG_IGN) do?

Show Explanation

Correct: B) Makes process ignore SIGINT (Ctrl-C won't interrupt).

Q48 Medium

What does alarm(5) do?

Show Explanation

Correct: A) Kernel sends SIGALRM after 5 seconds. Only one alarm at a time.

Q49 Easy

What does pause() do?

Show Explanation

Correct: C) Suspends process until any signal is delivered.

Q50 Hard

What happens with this code?

signal(SIGALRM, handler);
alarm(3);
pause();
Show Explanation

Correct: B)

Install handler → schedule alarm → wait in pause() → after 3 seconds SIGALRM arrives → handler runs → pause() returns.
