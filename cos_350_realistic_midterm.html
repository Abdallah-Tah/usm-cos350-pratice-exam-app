Midterm Questions

Question 1

How do you view the manual page for the command find? What does that command do?

Show answer

man find

This opens the manual page for the find command.

Question 2

Write a command that finds all words in file words that start with pre.

Show answer
grep '^pre' words

^ means start of line.

Question 3

You want a file to have these permissions: user read/write, group read, other read. Give the numeric chmod command.

Show answer
chmod 644 filename
Question 4

What system call can be used to get file information, including file size and permissions?

Show answer

stat()

Question 5

What field of struct stat stores the file size in bytes?

Show answer

st_size

Question 6

What numeric mask represents the other read permission bit?

Show answer

0004 in octal.

Question 7

Explain the difference between a hard link and a symbolic link.

Show answer

A hard link is another name for the same inode and same file data. A symbolic link is a separate file containing a path to another file.

Question 8

What is wrong with this code?

int a[10];
a[10] = 5;
Show answer

The valid indexes are 0 through 9. Writing to a[10] is out of bounds and causes undefined behavior.

Question 9

Why is int often used with getchar() instead of char?

Show answer

Because getchar() must hold both character values and the special value EOF.

Question 10

What does this declaration mean?

int main(int argc, char *argv[])
Show answer

argc is the number of command-line arguments, and argv is an array of strings containing them.

Question 11

Which function converts the string "42" to integer 42?

Show answer

atoi()

Question 12

What is the difference between open() and fopen()?

Show answer

open() is a system call that returns a file descriptor. fopen() is a C library function that returns a FILE * and uses buffering.

Question 13

What does read() return on end of file? What does it return on error?

Show answer

It returns 0 on end of file and -1 on error.

Question 14

What does lseek() do?

Show answer

It repositions the current file offset for a file descriptor.

Question 15

Explain the four parameters of qsort().

qsort(base, nmemb, size, compar);
Show answer

base is the start of the array, nmemb is the number of elements, size is the size of each element in bytes, and compar is the comparison function.

Question 16

Fix this function so it changes the caller's variable:

void change(int x) {
    x = x + 10;
}
Show answer
void change(int *x) {
    *x = *x + 10;
}

The caller must pass &var.

Question 17

Write your own version of strlen().

Show answer
int my_strlen(char *s) {
    int len = 0;
    while (*s != '\0') {
        len++;
        s++;
    }
    return len;
}
Question 18

What signal is usually sent by Ctrl-C? What is the default signal sent by kill pid?

Show answer

Ctrl-C usually sends SIGINT. kill pid usually sends SIGTERM.

Question 19

What do these calls do together?

signal(SIGALRM, wakeup);
alarm(3);
pause();
Show answer

The program installs a handler for SIGALRM, schedules the alarm in 3 seconds, and then waits in pause() until the signal arrives.

Question 20

What does it mean that Unix treats devices like files?

Show answer

It means many devices can be accessed using the same basic interface as files, such as open(), read(), write(), and close().
